CountryInfo

Search For Any Country

Extra Features:

1. Memoized(DP) Approach is taken into account, only one API call is needed for one query and then stored in session storage for later access. Latency Time reduced.

2. Error handling done for edge cases, in case no result for the provided query. Error handling for 404 status.

3. Bootstrap is used to style the web app

4. Code is commented

4. Hosted on Google Firebase.

Tasks Done:

1. The first page should have a search bar on top that should be used to search for any country by making an API call. You can use APIs from https://restcountries.eu/. :Done

2. The search results should be displayed on the same page as cards with the name and the flag of the country, without refreshing the web page. :Done

3. On clicking a card, the app should redirect to a page that shows country details. Display all interesting details that are returned by the API. You are free to design this page as you see fit. :Done

4. The details page should have a link to go back to the previous page, where the search results were displayed. :Done

5. Clean code and comments are highly appreciated. :Done

6. Bonus points for good error handling. :Done

7. Bonus points for adding interesting features as well. :Done
